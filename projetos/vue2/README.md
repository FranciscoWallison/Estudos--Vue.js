# Estudos Vue.js

> A Vue.js project

## Build Setup

``` bash
# install dependencies

npm install vue-cli@2.7.0 -g

# new project
vue init webpack-simple blog

# serve with hot reload at localhost:8080
npm run dev

# build for production with minification
npm run build
```
